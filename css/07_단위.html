#절대 길이 단위

  • px : 필셀(화소)
  • mm : 밀리미터
  • cm : 센치미터
  • in : 인치

# 상대 길이 단위

  • em : 현재 요소에 적영된 font-size에 비례한 크기 조절
    ex> 2.0em 일때 현재 요소의 font-size의 2배 크기
  • rem : 현재 페이지의 전체(html)에 적용된 font-size에 비례한 크기 조절
    ex > 2.0rem 일때 현재 페이지 전체 font-size의 2배 크기
  • vh : viewport width 현재 웹페이지를 보고있는 프로그램의 너비에 비례한 크기조절
    ex > 1vw는 viewport width의 1%크기
  • vh : viewport height viewport에 높이에 비례한 크기 조절
    ex > 1vh는 viewport heigth의 크기 1%크기
