# Psudo Class

  • 특정 상황만 선택하여 스타일을 적용하는 기능
  • 선택자 뒤에 :을 붙여 원하는 상황을 선택한다
  • ex> 마우스가 올려져 있는 상황, 해당 요소가 첫 번째 자식인 상황 등..

# 자수 사용되는 가상 클래스들

  • first-child, nth-child(An+B,even,odd,n), last-child
  • first-of-type, ntn-of-type(An+b,even,odd,n)last-of-type
  • hover : 마우스가 올려져 있는 상황
  • link : 링크
  • active : 활성화된 링크
  • visited : 방문했던 링크
