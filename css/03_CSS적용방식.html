# 페이지에 스타일시트를 적용하는 방법 3가지

1.inline CSS
- 줄 내부에서 CSS를 적용하는 방식
- 해당 요소의 style 속성에 스타일을 직접 정의하는 방식
- CSS 적용 우선 순위가 가장 높다
2.internal CSS
- 페이지 내부에서 CSS를 적용하는 방식
- 해당 페이지의 style 태그에 스타일을 직접 정의하는 방식
- 더 나중에 정의된 스타일시트가 앞의 시트를 덮어쓴다
3.external CSS
- 외부의 .css 파일을 불러와 페이지에 적용하는 방식
- link 태그를 사용해 외부의 css 파일을 불러올수 있다.
- 더 나중에 정의된 스타일시트가 앞의 시트를 덮어쓴다
