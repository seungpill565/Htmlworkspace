# Margin

  • 해당 요소와 다른 요소와의 거리를 결정한다
  • top margin과 bottom margin이 만나는 곳에서는 둘중 더 높은것만 적용된다
  • margin을 auto로 설정하면 해당 요소를 가운데로 정렬해준다.
