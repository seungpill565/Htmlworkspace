#CSS Selector

  • CSS는 선택자를 통해 어느 요소에 스타일을 적용할지 결정한다
  • tagname : 해당 태그인 모든 요소에 스타일을 적용한다
  • #id : 해당 id를 가진 요소를 선택해 스타일을 적용한다
  • .class : 해당 class를 가진 모든 요소를 선택해 스타일을 적용한다
tagName : div, id : none, className : none
tagName : div, id : div2, className : none
tagName: div, id : none, className : important
