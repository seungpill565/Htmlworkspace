# Javascript Html Dom

  • 자바스크립트는 HTNL 코드를 해석하여 document 객체를 생성한다
  • 자바스크립트 개발자는 DOM을 통해 html의 모든 요소에 접근할 수 있다

#Dom에서 원하는 요소를 찾는 메서드들

  • document.getElementById(id)
  • document.getElementbyTagName(tagName)
  • document.getElementsByClassName(className)
  • document.querySelector(CSS Selector)
  • document.querySelectorAll(CSS Selector)

# HTML 요소를 변경하는 방법들

  • elenebt.innerHTML : 요소 내부의 내용에 접근(태그 적용)
  • elenebt.innerTEXT : 요소 내부의 내용에 접근(태그 미적용,악용 방지)
  • element.attribute : 요소의 속성에 직접 접근
  • element.children : 해당 요소의 자식 요소 배열을 반환
  • element.setAttribute() : 요소의 속성을 메서드를 통해 수정
  • element.getAttribute() : 요소의 해당 속성을 문자열로 리턴받음
  • element.addEventListner(I) : 요소의 원하는 이벤트를 메서드를 통해 추가
