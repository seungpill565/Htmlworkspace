# 자바 스크립트로 웹 브라우저 직접 제어하기

  • 자바스크립트는 웹 브라우저와 소통하기 위한 객체 (Browser Object Model)을 통해 웹 브라우저를 직접 제어할 수 있다
  • window 객체는 사용자가 현재 사용중인 웹 브라우저를 의미한다
  • window 객체는 생랼이 가능하다(원래는 document 도 window.document다)

# Browser Object Model

  • Window : 웹 브라우저의 최상위 객체
  • Window.screan : 사용자의 모니터 정보 객체
  • window.history : 웹 브라우저의 페이지 이동 내역 객체
  • window.document : 현재 보고 있는 페이지의 DOM
  • window.location : 현재 보고 있는 위치(url)에 대한 정보를 담은 객체
  • window.navigator : 웹 브라우저의 정보를 담은 객체
